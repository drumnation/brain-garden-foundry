## What We Built

### Core Architecture
- **LangGraph orchestration** (`langgraph_executor.py`) - 22KB
  - State graph with conditional routing
  - SQLite checkpointing
  - Human-in-the-loop gates
  - Parallel execution with Send

- **Context injection** (`context_injection.py`) - 8.5KB
  - Neo4j queries for dependency/pattern/decision/state context
  - Token budget management (5K max)
  - Compression logic for oversized context

- **Story completion writer** (`story_writer.py`) - 9.5KB
  - Parses WHAT/HOW/DECISIONS/LANDMINES/LEARNINGS
  - Writes to Neo4j for future context retrieval
  - Creates pattern, decision, and landmine nodes

- **Prompt builder** (`prompt_builder.py`) - 12KB
  - Assembles spawn prompts
  - Embeds GROVE methodology
  - Includes borged Cortex patterns
  - Retry/escalation prompt variants

- **Verifier** (`verifier.py`) - 10KB
  - Evidence-based scoring (File 30pts, Test 40pts, Build 20pts, Deliverables 10pts)
  - 90+ to pass, 60-89 warning, <60 blocked

- **Ralph loop** (`ralph_loop.py`) - 12KB
  - 5-tier escalation (same retry → different agent → human → ralph)
  - Error classification
  - Failure pattern analysis

- **Discord reporter** (`discord_reporter.py`) - 9KB
  - Webhook-based status updates
  - Start/complete/fail/escalation embeds

## Key Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Orchestration | LangGraph | 60% less code, checkpointing, human-in-loop |
| Memory | Neo4j | Graph queries for context carryover |
| Patterns | Embedded in prompts | No ~/.claude dependency |
| Token budget | 5K context max | Prevents bloat, leaves room for task |
| Escalation | 5-tier Ralph loop | Graceful degradation |

**How It Works:**
1. Story completes → `grove_memory.record_story()` → Neo4j + DomainStore
2. Next story starts → `get_context_for_task()` → returns relevant past work
3. Story queued → `enqueue_story()` → Redis priority queue
4. Agent picks up → `dequeue_story()` → gets next story
5. Load distributed → `select_execution_node()` → picks least loaded node
