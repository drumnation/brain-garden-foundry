## 📦 How to Use This Generator

1. **Copy generator to your project:**
```bash
cp -r .generator/appwrite-deployment/ your-project/.generator/
```

2. **Configure settings:**
```bash
cd your-project/.generator/appwrite-deployment/
cp generator-config.example.json generator-config.json
# Edit generator-config.json with your settings
```

3. **Generate deployment commands:**
```bash
# Manual with Handlebars CLI
npx handlebars templates/orchestrator.md.hbs \
  -d generator-config.json \
  -o ../../.claude/commands/

# Or with Plop if configured
pnpm plop appwrite-deployment
```

4. **Execute orchestrator:**
```bash
# In Claude Code
/{projectName}-deploy
```
