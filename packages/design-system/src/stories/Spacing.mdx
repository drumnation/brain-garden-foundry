## Usage

```tsx
import { spacing, componentSpacing } from '@bg-kit/design-system/tokens';

// Base spacing
<Stack gap={spacing.md}>
  <div>Item 1</div>
  <div>Item 2</div>
</Stack>

// Component spacing
<Card padding={componentSpacing.componentPadding.lg}>
  <Stack gap={componentSpacing.componentGap.md}>
    <Title>Card Title</Title>
    <Text>Card content</Text>
  </Stack>
</Card>
```

## 8px Grid System

All spacing values are multiples of 8px, creating a harmonious vertical rhythm:

- 8px base unit for tight spacing
- 16px (2×) for default spacing
- 24px (3×) for comfortable spacing
- 32px (4×) for generous spacing
- 48px (6×) for section separation
