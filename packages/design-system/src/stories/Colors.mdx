## Color Roles

Functional color assignments for common UI elements:

- **Primary**: `{colorRoles.primary}` - Primary brand color
- **Secondary**: `{colorRoles.secondary}` - Secondary brand color
- **Text**: `{colorRoles.text}` - Body text color
- **Background**: `{colorRoles.background}` - Page background
- **Border**: `{colorRoles.border}` - Default border color
- **Disabled**: `{colorRoles.disabled}` - Disabled state color

## Usage

```tsx
import { colors } from '@bg-kit/design-system/tokens';

// Using semantic colors
<Button style={{ backgroundColor: colors.success.main }}>
  Success Button
</Button>

<Alert style={{ color: colors.error.main }}>
  Error message
</Alert>
```

## Accessibility

All color combinations meet WCAG 2.1 AA contrast requirements:
- Main + Contrast: At least 4.5:1 contrast ratio
- Light variants: For backgrounds
- Dark variants: For emphasis
