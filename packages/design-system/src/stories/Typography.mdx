## Font Weights

- **Regular (400)**: {typography.fontWeight.regular}
- **Medium (500)**: {typography.fontWeight.medium}
- **Semibold (600)**: {typography.fontWeight.semibold}
- **Bold (700)**: {typography.fontWeight.bold}

## Line Heights

- **Tight**: {typography.lineHeight.tight} - For headings
- **Normal**: {typography.lineHeight.normal} - For body text
- **Relaxed**: {typography.lineHeight.relaxed} - For readability

## Usage

```tsx
import { textStyles } from '@bg-kit/design-system/tokens';

<h1 style={textStyles.h1}>Page Title</h1>
<p style={textStyles.body}>Body content</p>
<label style={textStyles.label}>Input Label</label>
```
